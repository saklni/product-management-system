# Product Management System

Product Management System adalah aplikasi manajemen produk berbasis web yang dibangun menggunakan Next.js dan TypeScript.

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* MongoDB Atlas
* Auth.js
* Cloudinary

## Features

* User Authentication
* Product CRUD
* Product Image Upload
* Dashboard
* Kasir 
* Riwayat transaksi
